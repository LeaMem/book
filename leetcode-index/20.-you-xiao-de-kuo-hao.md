# 20. 有效的括号

> Difficulty: Easy

 给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串，判断字符串是否有效。

有效字符串需满足：

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。

注意空字符串可被认为是有效字符串。

 **示例 1:**

```text
输入: "()"
输出: true
```

 **示例 2:**

```text
输入: "()[]{}"
输出: true
```

 **示例 3:**

```text
输入: "(]"
输出: false
```

 **示例 4:**

```text
输入: "([)]"
输出: false
```

 **示例 5:**

```text
输入: "{[]}"
输出: true
```

bool valid(char * stack, int *index, char left){

    if(*index >= 0 && stack[*index] == left){
        (*index)--;
        return true;
    }
    return false;
}

bool isValid(char *s) {
    int len = strlen(s);
    char stack[8000];
    int top = -1;
    for(int i = 0; i < len; i++){

        if(s[i] == ')' && !valid(stack, &top, '(')){
            return false;
        }

        if(s[i] == '}' && !valid(stack, &top, '{')){
            return false;
        }

        if(s[i] == ']' && !valid(stack, &top, '[')){
            return false;
        }

        if(s[i] == '[' || s[i] == '{' || s[i] == '('){
            stack[++top] = s[i];
        }
    }


    return top < 0;

}
